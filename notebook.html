topicexplorer notebook¶

Launches a web server with the Jupyter Notebook server.

Populates a directory with a tutorial notebook, prepopulated with commands to load the current dataset.

Command Line Arguments¶

No Launch (--no-launch)¶

Starts the notebook server without opening the notebook interface in the default browser.
